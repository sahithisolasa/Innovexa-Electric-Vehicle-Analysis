# Electric Vehicle Adoption Analysis and Range Prediction

## Innovexa Catalyst 2026 - Project 1

### Project Overview

This project focuses on analyzing Electric Vehicle (EV) population data to understand adoption trends, manufacturer dominance, vehicle characteristics, and geographic distribution. The project also implements a machine learning model to predict electric vehicle range based on vehicle attributes.

---

## Objective

* Analyze electric vehicle adoption trends.
* Understand manufacturer and vehicle-type distribution.
* Explore geographic patterns of EV registrations.
* Visualize important insights using charts and graphs.
* Build a machine learning model to predict electric vehicle range.

---

## Dataset Information

The dataset contains information about registered electric vehicles, including:

* Vehicle Manufacturer
* Vehicle Model
* Model Year
* Electric Vehicle Type
* Electric Range
* County and City
* Electric Utility Information
* CAFV Eligibility

Dataset Size:

* Rows: 279,780
* Columns: 16

## Key Findings

1. Battery Electric Vehicles (BEVs) represent the majority of registered electric vehicles.

2. Tesla dominates the EV market and significantly outperforms other manufacturers in total registrations.

3. Electric vehicle adoption has increased rapidly since 2020.

4. King County has the highest concentration of EV registrations.

5. Electric vehicle range varies considerably across manufacturers.

6. The machine learning model achieved excellent predictive performance.

---

## Model Performance

| Metric                         | Value  |
